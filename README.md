# Théorie — A Theory of Scent
### A Niche Perfume Brand Landing Page | Business Website Midterm Exam
---
## 📌 Project Overview
This is a business landing page built to showcase **Théorie**, a fictional niche perfume brand based in Manila. Instead of a basic product page, this site presents a refined digital experience — exploring the brand's identity, fragrance collection, philosophy, and an original interactive game.

The goal of this project is to create an **elegant, brand-driven web layout** while practicing and mastering Tailwind CSS fundamentals, Flexbox, and CSS Grid from scratch.

## 📄 Sections
The website is structured as a **Single-Page Application (SPA)** — all content flows seamlessly on one page and is navigated through a sticky top navbar.

1. **Hero Section** — Full-viewport introduction to Théorie with a gradient background, tagline, and dual CTAs.
2. **What Sets Us Apart** — Three feature cards highlighting the brand's competitive edge: Handcrafted Formulas, Lasting Presence, and Curated Ingredients.
3. **Our Fragrances** — A four-card collection showcase (No. 01–04) with scent family tags and individual fragrance descriptions.
4. **Mission & Vision** — Split layout detailing the brand's mission to make scent personal and its vision to lead the Southeast Asian niche fragrance market.
5. **The Nose Knows** — An original interactive Simon Says memory game using vanilla JavaScript, themed around perfume ingredients (Rose, Bergamot, Cedar, Musk, Vanilla, Jasmine, Amber, Sandalwood).
6. **FAQ** — Accordion-style frequently asked questions covering ingredients, concentration, cruelty-free practices, storage, and returns.

## 🎮 Interactive Game — The Nose Knows
A Simon Says memory game themed around fragrance ingredients. A sequence of ingredient cards flashes on screen — the player must repeat the sequence in the correct order. Each successful round adds one more ingredient to the sequence, and the game speeds up over time. A wrong selection triggers a red flash game-over state. The game tracks Level, Score, and Best Score.

## 🎨 Design
- **Theme:** Niche Luxury Fragrance / Editorial Brand
- **Color Palette:** Linen, Forest Green, Bark (near-black), Sage, Amber, and Clay (earthy, warm, and sophisticated)
- **Typography:** Playfair Display (for elegant display headings) + Outfit (for clean, modern body text)
- **Layout:** Fully responsive using Tailwind's utility classes — optimized for desktop, tablet, and mobile viewing.

2. Ensure you have an active internet connection (for CDN resources).
